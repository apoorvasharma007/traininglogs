## How to Run the Code

### Initialize Database
```bash
python scripts/init_db.py
```
- Creates `traininglogs.db` with full schema
- Runs all migrations
- Safe to run multiple times (idempotent)

### Run CLI Application
```bash
python -m src.cli.main
```
- Starts interactive workout logging
- Creates sessions, logs exercises
- Saves to database

### Test Imports (No Circular Dependencies)
```bash
python -c "from src.cli.main import main; print('‚úì')"
python -c "from src.persistence import Database; print('‚úì')"
```

### Run Safety Verification
```bash
python .agent/scripts/verify_changes.py
```
- Checks circular imports
- Tests database init
- Validates major import paths
- Verifies file structure
